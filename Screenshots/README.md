# 🏥 Clinic Management System

A desktop-based Clinic Management System developed using **C#**, **Windows Forms**, **.NET 9**, and **Microsoft SQL Server** following the **Three-Tier Architecture (Presentation Layer, Business Logic Layer, Data Access Layer)**.

This project was developed as part of a university Software Engineering course to simplify clinic operations by managing doctors, patients, appointments, invoices, and users through a secure and organized desktop application.

---

## ✨ Features

- 🔐 Secure Login System
- 👤 Role-Based Access (Admin / Receptionist)
- 👨‍⚕️ Doctor Management
- 🩺 Patient Management
- 📅 Appointment Management
- 🚫 Appointment Conflict Prevention
- 💰 Automatic Invoice Generation
- 🖨️ Invoice Printing
- ✔️ Input Validation
- 🔍 Search Functionality

## 🚀 Getting Started

### Requirements

- Windows 10 or later
- .NET 9 SDK
- Microsoft SQL Server
- Visual Studio 2022

### Installation

1. Clone the repository.
2. Restore the database using the SQL script inside the **Database** folder.
3. Update the connection string in `App.config`.
4. Open the solution in Visual Studio.
5. Build and run the project.
